# SwipeClock Demonstration

A demonstration of using React to create a simple single page application.

## Requires

* node.js
* yarn

## Build and Run

```
yarn
yarn global add json-server
```

In a new shell:
```
cd root/data
json-server db.json
```

In another shell
```
cd root/
yarn run dev
```

In a client
```
http://localhost:8080
```
